Prompt Station Manual

使用手冊 (V6.3)

歡迎使用 Prompt Station。本手冊將協助您快速掌握 V6.3 的進階功能。

1. 快速收集靈感 (Right-Click Inbox)

在瀏覽任何網頁時，看到想要保存的 Prompt 或文案：

  1. 選取該段文字。
  2. 點擊滑鼠右鍵，選擇 「Save to Prompt Station」。
  3. 該內容會立即儲存至擴充功能的 Inbox 區域。
  4. 打開擴充功能視窗，您可以將 Inbox 的內容移動到正式標籤庫中。

2. 匯出對話記錄 (Export Chat)

想備份 ChatGPT 或 Gemini 的對話嗎？

  1. 打開擴充功能視窗。
  2. 點擊底部的 「Export Chat (.md)」 按鈕。
  3. 系統會自動偵測當前頁面的對話內容，並下載為 Markdown 檔案。

注意：支援 ChatGPT, Gemini, Copilot 及 Local AI 介面。

3. Local AI 與自訂域名

若您使用本地部署的 AI (如 Ollama, LM Studio) 或自架 WebUI：

  1. 在擴充功能底部找到 「Local AI / Custom Domain」。
  2. 輸入您的 WebUI IP 位址或網域 (例如 localhost:3000 或 192.168.1.50)。
  3. 點擊 Add。
  4. 現在，Prompt Station 也能在該網站上運作了！

常見問題 (FAQ)

Q: 資料存在哪裡？會上傳雲端嗎？

A: 不會。所有資料 100% 儲存在您的瀏覽器本地端 (Chrome Storage Local)。我們重視您的隱私。

Q: 為什麼右鍵選單沒出現？

A: 請確認您已經「選取了文字」。為了保持選單整潔，只有在選取文字時該選項才會出現。

User Manual (V6.3)

Welcome to Prompt Station. This guide covers the advanced features of version 6.3.

1. Right-Click Capture (Inbox)

Found a great prompt while browsing?

  1. Select the text.
  2. Right-click and select "Save to Prompt Station".
  3. The text is instantly saved to your Inbox.
  4. Open the extension popup to manage or categorize items from your Inbox later.

2. Export Chat History

Need to backup your conversation?

  1. Open the extension popup.
  2. Click the "Export Chat (.md)" button at the bottom.
  3. The extension scrapes the current conversation and downloads it as a Markdown file.

Supported: ChatGPT, Gemini, Copilot, and Local AI interfaces.

3. Local AI & Custom Domains

Using Ollama, LM Studio, or a private WebUI?

  1. Scroll to "Local AI / Custom Domain" in the popup.
  2. Enter your IP or domain (e.g., localhost:3000 or 192.168.1.50).
  3. Click Add.
  4. Prompt Station will now inject scripts into that domain to enable functionality.

FAQ

Q: Where is my data stored?

A: 100% locally in your browser (Chrome Storage Local). No cloud sync, no tracking.

Q: Why don't I see the right-click menu?

A: Ensure you have selected text. The menu item is hidden when no text is selected.
