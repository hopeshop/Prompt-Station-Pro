Right-Click Inbox

Capture ideas without breaking your flow:

  1. Highlight any text on a webpage.
  2. Right-click and select "Save to Prompt Station".
  3. The text is instantly saved to your Inbox.
  4. Open the extension popup to manage or categorize items later.

Export Chat History

Backup your conversations to Markdown:

  1. Open the extension popup on ChatGPT, Gemini, or Copilot.
  2. Click the "Export Chat (.md)" button at the bottom.
  3. The system scrapes the current thread and downloads a .md file.

Local AI & Custom Domains

Using Ollama, LM Studio, or a private WebUI?

  • 1 Click "Local AI / Custom Domain" in the popup.
  • 2 Enter IP or domain (e.g. localhost:3000).
  • 3 Click Add. The extension is now active on that site.

Performance Optimization

How to achieve optimal local AI performance:

  • 1 Control Memory Size: Large code blocks (e.g. 1000 lines) in fixed memory significantly increase response time. Only include content relevant to current task.
  • 2 Use Appropriate Models: Qwen2.5-Coder 14B is optimal for coding, achieving 2-3s responses on RTX 5060 Ti 16GB.
  • 3 Clean Memory Regularly: Use "AI Summarize" to condense dynamic memory and keep it lean.
  • 4 Auto Context Detection: System auto-detects model context sizes (Qwen2.5: 16K, Llama 3.1: 32K) for optimization.

FAQ

Where is my data stored?
Locally. We adhere to a Local-First policy. All data resides in your browser's Chrome Storage Local. No cloud sync.
Why don't I see the context menu?
The menu item is context-aware. It only appears when you have highlighted/selected text on the page.
What does "..." mean in Dynamic Memory?
It's a preview truncation indicator. The full memory content is still there, just truncated in display to save space. AI uses the complete content during operation.
Local AI response is slow, what to do?
Main reason is too much content in fixed memory. Try clearing or reducing fixed memory content. Tests show 2-3s response on RTX 5060 Ti 16GB with empty memory, but 1000 lines of code increases it to 20+ seconds.
Does export include memory content?
Yes! Clicking "Export Chat (.md)" exports the complete conversation history, including fixed memory and dynamic memory content.
